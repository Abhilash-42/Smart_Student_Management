# Smart_Student_Management
A Student Academic & Attendance Management System developed using Python, Flask, and MongoDB.
This project helps manage student academic records, marks, attendance, and performance analytics through a simple web dashboard interface.

The system allows teachers or administrators to add student records, view performance, generate reports, and visualize student performance using charts.
# Features
Add student details and subject marks

Store student records using MongoDB database

Automatic percentage calculation

View all students in a structured table

Delete student records

Generate student PDF reports

Performance visualization using charts

Bar chart for student percentage comparison

Pie chart for performance distribution

Clean dashboard UI with CSS styling
# Technologies Used
Backend: Python

Web Framework: Flask

Database: MongoDB

Frontend: HTML, CSS, JavaScript

Charts: Chart.js

PDF Generation: ReportLab
